# telegram-cli
Telegram c++ cli

[![Build Status](https://travis-ci.com/cli-inc/telegram-cli.svg?branch=master)](https://travis-ci.com/cli-inc/telegram-cli) 

# Build

Clone GitHub Repository

     git clone --recursive https://github.com/vysheng/tg.git && cd tg

Build

    cmake
    make

Install

     sudo make install
